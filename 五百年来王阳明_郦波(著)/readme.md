# 五百年来王阳明 读书笔记

## 推荐语
阳明心学是浮躁现实里最好的心灵解药，恶劣环境中强大的精神武器。
心外无物，心即理；知行合一，事上练；格物致知，致良知；
阳明心学入门第一书，郦波教授结合两千年来中华文明发展史，生动讲述了王阳明的传奇人生和心学思想的创建历程，悬念迭起，一气呵成。
